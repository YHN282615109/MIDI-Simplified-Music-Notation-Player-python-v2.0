
# 🎵 MIDI简谱播放器 V2.0

[![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)](https://gitee.com/YHN282615109)
[![Python](https://img.shields.io/badge/python-3.7+-green.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/license-MIT-yellow.svg)](LICENSE)

一款功能强大的MIDI简谱播放与编辑软件。支持16个独立音轨，可导入标准MIDI文件、实时播放简谱并导出MIDI文件，同时支持将简谱导出为精美图片。

---

## ✨ 核心亮点

- 🎼 **MIDI文件导入** - 导入标准MIDI文件，自动转换为简谱文本（高精度转换）
- 🎹 **多音轨支持** - 16个独立音轨，MIDI多音轨可分别导入到不同音轨
- 🖼️ **简谱图片导出** - 将编辑好的简谱导出为PNG图片（普通版+专业版）
- 🔧 **音轨独立设置** - 每个音轨可单独设置调式、乐器、速度
- 🎵 **完整MIDI输出** - 支持实时MIDI播放和MIDI文件导出
- 🎲 **随机谱曲** - 自动生成随机旋律，激发创作灵感

## 🚀 快速开始

### 系统要求

- Windows 7/10/11 | macOS | Linux
- Python 3.7 或更高版本
- MIDI输出设备（虚拟或硬件）

### 安装步骤

#### 1. 克隆或下载项目

```bash
git clone https://gitee.com/YHN282615109/midi-player.git
cd midi-player
```

#### 2. 安装依赖

```bash
pip install pygame mido pillow
```

#### 3. 运行程序

```bash
python midi_player_v2.py
```

## 📝 简谱格式说明

### 头部格式（每个音轨的第一行）

```
调式,乐器编号,速度
```

**示例**：
- `G 大调,40,200` → G大调，小提琴音色，速度200ms/拍
- `C 大调,0,500` → C大调，钢琴音色，速度500ms/拍

### 支持的调式

| 调式 | 调式 | 调式 |
|------|------|------|
| C 大调 | C# 大调 | D 大调 |
| D# 大调 | E 大调 | F 大调 |
| F# 大调 | G 大调 | G# 大调 |
| A 大调 | A# 大调 | B 大调 |

### 乐器编号参考

| 编号 | 乐器 | 编号 | 乐器 |
|------|------|------|------|
| 0 | 大钢琴 | 10 | 音乐盒 |
| 1 | 明亮的钢琴 | 40 | 小提琴 |
| 2 | 电钢琴 | 41 | 中提琴 |
| 3 | 酒吧钢琴 | 42 | 大提琴 |
| 4 | 柔和的电钢琴 | 48 | 弦乐合奏 |
| 5 | 电钢琴加合唱 | 56 | 小号 |
| 6 | 羽管键琴 | 57 | 长号 |
| 7 | 击弦古钢琴 | 58 | 大号 |

### 音符语法

| 语法 | 含义 | 示例 |
|------|------|------|
| `1-7` | 基本音符（do-re-mi-fa-sol-la-si） | `5` = sol |
| `'` | 高八度 | `1'` = 高音do |
| `.` | 低八度 | `1.` = 低音do |
| `#` | 升号 | `5#` = 升sol |
| `-` | 延长时值（1.5倍） | `5--` = 延长两次 |
| `0` | 休止符 | `0` = 休止一拍 |
| `[ ]` | 音符组（占一拍） | `[123]` = 三个音符平分一拍 |
| `( )` | 音符组（时值减半） | `(56)` = 拍长减半 |
| `\|` | 小节分隔符（可选） | `1\|2\|3` |

### 完整示例

```
G 大调,40,200
5-351'--76-1'-5---5-123-212----5-351'--76-1'-5---5-234--7.1----6-1'-1'---7-671'---671'665312----5-351'--76-1'-5---5-234--7.1------
```

> 表示：G大调，小提琴音色，200ms/拍，演奏《小星星》变奏曲

## 🎵 调式移调原理

程序会根据设置的调式自动将简谱音符转换为正确的MIDI音高：

| 调式 | 移调值 | 说明 |
|------|--------|------|
| C 大调 | +0 | 基本调式 |
| G 大调 | +7 | 升5度 |
| D 大调 | +2 | 升2度 |
| A 大调 | +9 | 升6度 |
| E 大调 | +4 | 升3度 |
| B 大调 | +11 | 升7度 |
| F 大调 | -5 | 降4度 |

## ❓ 常见问题

### Q1: 没有声音怎么办？

**A:** 请检查：
1. 是否安装了MIDI输出设备（推荐 VirtualMIDISynth）
2. 可在"设置/选项"→"MIDI设备设置"中切换设备
3. Windows用户可在"设置/选项"中切换MIDI设备

### Q2: 如何安装虚拟MIDI设备？

**A:**
- **Windows**: 下载安装 [CoolSoft VirtualMIDISynth](https://coolsoft.altervista.org/en/virtualmidisynth)
- **macOS**: 使用系统自带的 IAC Driver（音频MIDI设置中启用）
- **Linux**: 使用 Timidity 或 Fluidsynth

### Q3: 如何导入MIDI文件？

**A:**
1. 点击菜单"文件"→"导入MIDI文件（多音轨）"
2. 选择MIDI文件
3. 在弹出的窗口中选择要导入的音轨（可多选）
4. 选择导入到当前音轨或依次导入到多个音轨

### Q4: 简谱播放不正确？

**A:** 请检查：
1. 头部格式是否正确 `调式,乐器编号,速度`
2. 音符写法是否符合规范（数字1-7，支持#、.、'）
3. 是否有未闭合的括号 `[ ]` 或 `( )`

### Q5: 如何导出简谱图片？

**A:**
1. 点击菜单"文件"→"导出简谱图片（简单版）"或"导出简谱图片（专业版）"
2. 选择保存位置
3. 程序自动为当前有内容的音轨生成PNG图片

### Q6: 速度值如何理解？

**A:** 速度值表示每拍的毫秒数：
- 数值**越小**播放**越快**（如100 = 10拍/秒）
- 数值**越大**播放**越慢**（如800 = 0.75拍/秒）
- 建议范围：100-800

### Q7: 软件卡顿怎么办？

**A:**
1. 关闭其他占用CPU的程序
2. 减少同时播放的音轨数量
3. 适当降低BPM值
4. 避免在播放时编辑乐谱
